## 🚀 About The Project

**Arcade** (internally referenced as *CRYSTAL*) is a web-based application focused on delivering a seamless, responsive, and minimalist user experience. The project serves as a foundational frontend architecture that can be easily extended and customized. Hosted on Vercel, it leverages automated CI/CD pipelines to ensure the live environment is always synchronized with the latest main branch commits.

---

## ✨ Features

- **Minimalist UI/UX:** Clean design language utilizing deep contrasts and dark-themed aesthetics.
- **Lightning Fast Delivery:** Globally distributed edge deployment via Vercel for sub-second load times.
- **Modular Asset Management:** Strictly compartmentalized static assets and core logic for easier debugging.
- **Responsive Layout:** Adaptive styling ensuring uniform functionality across desktop and mobile devices.
- **Automated CI/CD:** Instant deployment triggers on every repository push.

---

## 💻 Technologies Built With

This project relies on core web technologies, keeping dependencies lightweight and execution fast:

- **Frontend Logic:** `JavaScript (ES6+)`
- **Markup:** `HTML5`
- **Styling:** `CSS3`
- **Scripting:** `Shell`
- **Hosting & CI/CD:** `Vercel`

## 🛠️ Getting Started

To get a local copy up and running locally, follow these simple steps.

### Prerequisites
Ensure you have the following installed on your local machine:
- [Git](https://git-scm.com/downloads)
- A modern web browser (Chrome, Firefox, Safari, Edge)
- *(Optional)* A local development server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code.

### Installation

1. **Clone the repository**
```bash
   git clone [https://github.com/shai-tan1/Arcade.git](https://github.com/shai-tan1/Arcade.git)
```
2. **Navigate into the project directory**
```
    cd Arcade
```

3. **Launch the application**
   - **Method A:** Simply double-click the `index.html` file inside the `main` directory to open it in your default browser.
   - **Method B:** If using VS Code, right-click `index.html` and select **"Open with Live Server"** to enable hot-reloading during development.
